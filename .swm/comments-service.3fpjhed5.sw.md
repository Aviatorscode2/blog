## **Goals**

1.  Create a comment associated with the given post ID

2.  Retrieve all comments associated with the given post ID

<br/>

|### **Path**         |### **Method**|### Body         |
|---------------------|--------------|-----------------|
|`/posts/:id/comments`|POST          |{content: String}|
|`/posts/:id/comments`|GET           |\-               |
